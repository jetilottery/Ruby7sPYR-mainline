Rubí 7

¡Que los rojos y brillantes rubíes de Rubí 7 te llenen de riquezas! Gana premios revelando símbolos de Rubí 7. ¡Consigue dos símbolos de Rubí 7 para duplicar el premio y tres para triplicarlo! Consigue 3 o más símbolos de bonificación de rubí especiales para activar el juego de bonificación, ¡con lo que podrás girar la ruleta y ganar más premios asombrosos!

Cómo apostar

Costo del boleto

Muestra el costo actual del boleto.

Presiona la flecha izquierda (-) para reducir el costo del boleto.

Presiona la flecha derecha (+) para aumentar el costo del boleto.

Presiona COMPRAR para descontar el costo del boleto de tu saldo y jugar.

Cómo jugar

Presiona cada símbolo de rubí para revelar símbolos de Rubí 7, de gemas y de rubí de bonificación. Revela símbolos de Rubí 7 para ganar un premio. Encuentra un súmbolo de Rubí 7 doble para duplicar el premio y triple para triplicarlo.

Revela 3 símbolos rubí de bonificación para activar el juego de bonificación. Presiona la flecha circular para girar la ruleta y ganar más premios. Gana un premio cuando el puntero se detenga en una casilla con un valor de premio. Los premios se acumulan hasta que termina el juego de bonificación. Cuando el puntero se detenga en una símbolo de flecha, avanzarás a la siguiente ruleta con mejores premios. Si se detiene en una X roja, se eliminará un rubí de bonificación. El juego de bonificación termina cuando no te queden más rubíes de bonificación.

Los premios de la bonificación se agregan al valor de premio total.

Presiona el mensaje del premio para ver el resultado y quitar la notificación.

Mostrar todos

Presiona MOSTRAR TODOS para completar el juego automáticamente. El juego continúa hasta que se revelen todos los símbolos y se entreguen todos los premios.

Reglas

El uso indebido o mal funcionamiento anulan todos los pagos y jugadas.

Los premios se muestran como monedas.

Las decisiones del jugador durante la partida no afectan el resultado del juego.

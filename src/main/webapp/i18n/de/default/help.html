RUBY 7s

Möge das strahlende, glanzvolle Rot von Ruby 7s Ihnen Reichtümer bringen! Gewinnen Sie Preise, indem Sie Ruby 7 Symbole aufdecken. Decken Sie zwei Ruby Ruby 7-en auf, um den Preis zu verdoppeln, und verdreifachen Sie den Preis mit drei Ruby 7-en! Wenn 3 oder mehr Spezialbonusrubin-Symbole aufgedeckt werden, wird das Bonusspiel ausgelöst. Sie erhalten dann die Chance, das Rad zu drehen und sogar noch mehr funkelnde Preise zu gewinnen!

So platzieren Sie einen Einsatz

Ticketpreis

Zeigt den aktuellen Ticketpreis an.

Drücken Sie auf den linken Pfeil (–), um den Ticketpreis zu verringern.

Drücken Sie auf den rechten Pfeil (+), um den Ticketpreis zu erhöhen.

Drücken Sie auf KAUFEN, um den Ticketpreis vom Guthaben abzuziehen und das Spiel zu spielen.

Spielanleitung

Drücken Sie auf jedes Rubinsymbol, um Ruby 7-, Edelstein- oder Bonusrubin-Symbole aufzudecken. Decken Sie beliebige Ruby 7-Symbole auf und gewinnen Sie einen Preis. Finden Sie ein doppeltes Ruby 7-Symbol und verdoppeln Sie damit den Preis. Mit einem dreifachen Ruby 7-Symbol verdreifachen Sie Ihren Preis.

Decken Sie 3 oder mehr Bonusrubin-Symbole auf, um das Bonusspiel auszulösen. Drücken Sie den kreisförmigen Pfeil, um das Rad und nutzen Sie die Chance auf den Gewinn weiterer Preise. Gewinnen Sie einen Preis, wenn der Zeiger auf einem Preis-Wert landet. Die Preise werden gesammelt, bis der Bonus endet. Steigen Sie zum nächsten Rad auf, wenn der Zeiger auf einem Pfeilsymbol landet. Wenn Sie auf einem roten X landen, wird ein Bonusrubin entfernt. Das Bonusspiel endet, wenn keine Bonusrubine mehr übrig sind.

Bonusgewinne werden zum Gesamtgewinnbetrag addiert.

Verwerfen Sie die Gewinnnachricht und schauen Sie sich das Spielergebnis an, indem Sie auf die entsprechende Nachricht drücken.

Alle aufdecken

Drücken Sie auf ALLE AUFDECKEN, um das Spiel automatisch abzuschließen. Das Spiel wird fortgesetzt, bis alle Auswahlen getroffen und Preise vergeben wurden.

Regeln

Bei Missbrauch oder Fehlfunktionen werden alle Gewinne und Spiele annulliert.

Gewinne werden in der ausgewählten Währung angezeigt.

Spielerentscheidungen, die während des Spiels getroffen werden, beeinflussen das Ergebnis des Spiels nicht.

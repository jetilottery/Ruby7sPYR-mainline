RUBY 7s

Låt det strålande, glänsande röda i Ruby 7s ge dig rikedomar! Vinn priser genom att avslöja Ruby 7-symboler. Avslöja en symbol med två Ruby 7:or för att dubbla priset och en symbol med trippel-Ruby 7:or för att trippla priset! När 3 eller fler speciella bonusrubin-symboler avslöjas aktiveras bonusspelet och ger dig chansen att snurra på hjulet och vinna ännu fler glittrande priser!

Så här satsar du

Lottkostnad

Visar den aktuella lottkostnaden.

Tryck på vänsterpilen (–) för att minska lottkostnaden.

Tryck på högerpilen (+) för att öka lottkostnaden.

Tryck på KÖP för att dra lottkostnaden från saldot och börja spela.

Så här spelar du

Tryck på varje rubinsymbol för att avslöja Ruby 7-symboler, ädelstensymboler eller bonusrubin-symboler. Avslöja en Ruby 7-symbol och vinn ett pris. Hitta en symbol med dubbel-Ruby 7:or för att dubbla priset och en symbol med trippel-Ruby 7:or för att trippla priset.

Avslöja 3 eller fler bonusrubin-symboler för att starta bonusspelet. Tryck på cirkelpilen för att snurra på hjulet för en chans att vinna fler priser. Vinn ett pris när pekaren stannar på ett prisvärde. Priserna ackumuleras tills bonusen är över. Gå vidare till nästa hjul när pekaren stannar på en pilsymbol. Landa på ett rött X så avlägsnas en bonusrubin. Bonusen avslutas när inga bonusrubiner återstår.

Bonusvinster läggs till det totala prisvärdet.

Avfärda vinstmeddelandet och se spelets utfall genom att trycka på meddelandet.

Avslöja alla

Tryck på AVSLÖJA ALLA för att spela klart spelet automatiskt. Spelet fortsätter tills alla val har gjorts och eventuella vinster har tilldelats.

Regler

Missbruk eller funktionsfel ogiltigförklarar alla utbetalningar och spel.

Vinster visas som valuta.

Val som gjorts av spelare under spelets gång påverkar inte spelets utfall.
